| 后缀字母 | 说明 | 例子 |
| ---  | --- | --- |
| or | 一般表示人 | operator , director |
| er | 表示人的职业的/...的人 | teacher 教师<br>employer 雇主 |
| ee | 被...的人 | employee 雇员 |
| ess | 表示女性 | hostess 女主人<br>waitress 女服务员 |
| man | 表示人 | policeman , postman |
