# 单数-复数的规则

1. 一般的名词 + s

2. 以 s、sh、x、ch 结尾的 + es

3. 以 f、fe、结尾的替换为 ves

4. 辅音字母 + y 结尾的名词，变 y 为 i + es

5. 元音 + o 结尾，+ s

6. 辅音 + o 结尾，+ es

## 1. 一般的名词 + s

| 单数 | 复数 |
| --- | --- |
| a girl | girls |
| an officer | officers |
| a coat | coats |

## 2. 以 s、sh、x、ch 结尾的 + es

| 单数 | 复数 |
| --- | --- |
| a bus | buses |
| a box | boxes |
| a brush | brushes |

## 3. 以 f、fe、结尾的替换为 ves

| 单数 | 复数 |
| --- | --- |
| a wife | wives |
| a shelf | shelves |
| a knife | knives |
| a leaf | leaves |

## 4. 辅音字母 + y 结尾的名词，变 y 为 i + es

| 单数 | 复数 |
| --- | --- |
| a baby | babies |
| a lady | ladies |

## 5. 元音 + o 结尾，+ s

| 单数 | 复数 |
| --- | --- |
| a zoo | zoos |
| a radio | radios |

## 6. 辅音 + o 结尾，+ es

| 单数 | 复数 |
| --- | --- |
| a hero | heroes |
| a potato | potatoes |
| a tomato | tomatoes |
