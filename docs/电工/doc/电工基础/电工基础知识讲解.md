# 电工基础知识讲解

## 1. 工厂用电

### 1.1  工厂一般有电源 380V 和 220V

**三相：** 黄绿红、蓝、双色，电压 380V

| 电线 | 颜色 |
| --- | --- |
| 三相 | 黄(R)、绿(S)、红(T) |
| 零线 | 蓝(N) |
| 地线 | 双色(PE) |

**单相：** 红(对应火线L)、黑(对应零线N)，左零右火

### 1.2 人体的安全电压是 36V。

### 1.3 变压器的符号：T

### 1.4 什么是电压、电流、电阻、功率

| 类型 | 单位 |
| --- | --- |
| 电压 U | V(伏安)、KV(千伏安) |
| 电流 I | mA(毫安)、A(安) |
| 电阻 R | Ω(欧姆)、kΩ(千欧) |
| 功率 P | W(瓦)、KW(千瓦) |

**换算公式：**

```txt
U = I * R

P = U * I * cosφ
```

> **cosφ 符号：** 功率因素
